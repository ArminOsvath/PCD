Prerequisites: make

How to run:

  0) You must be in the same directory as the makefile file 
  1) Open 2 terminals
  2) In either run the command `make`
  3) In one terminal run `make runs`
  4) In the other terminal run `make runc`
  5) `cd scripts`
  6) `for f in *.py; do python3 "$f"; done`
